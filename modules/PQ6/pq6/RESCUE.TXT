从应急磁盘运行软件

应急磁盘中包含引导计算机以及运行 DOS GUI 版本的软件所需的文件。
您必须从第一张应急磁盘进行引导。

在以下情况下，应急磁盘将很有用：

- 如果隐藏了安装软件的硬盘分区，而要回到软件中来取消隐藏该分区。

- 如果意外地将某个分区转换为 FAT32 而操作系统不支持 FAT32，
从而不能引导计算机。（使用应急磁盘将该分区再转换回 FAT。）

- 如果计算机只安装有 Windows NT 3.51，而您需要操作 NT 引导分区
  （带有 Windows 目录的分区）。

- 如果出现其它情况，而您无法访问 CD 或硬盘上的 Windows 版本的软件。

        
要使用应急磁盘，请：
------------------------

1. 从第一张应急磁盘引导计算机。

2. 在出现提示时插入第二张应急磁盘。

程序将自动运行。

尽管应急磁盘运行的是 DOS 版本的软件，但该版本是一个完整的 GUI 应用程序，
其功能与 Windows 可执行文件相似。

要访问联机帮助，请插入第二张应急磁盘，然后从“帮助”菜单中选择“常规”
或“索引”。

有关从应急磁盘运行软件的其它信息，包括应急磁盘版本的软件与 Windows 版本
有哪些不同，请参阅用户指南的第一章。
